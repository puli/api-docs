Class SyncTypeName

Synchronizes a type name with the discovery.

The method Puli\RepositoryManager\Discovery\Type\SyncTypeName::takeSnapshot() must be called before executing the operation. This method will record whether the type name is currently enabled (i.e. loaded in the discovery).

Once the operation is executed, another snapshot is taken. If the snapshots differ, type is either defined or undefined in the discovery, depending on the outcome.
