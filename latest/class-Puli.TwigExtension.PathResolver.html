public string
# resolvePath( string $path, string $currentDir )

Resolves a path depending on the current directory.

Resolves a path depending on the current directory.

If the path is a relative path, the Puli repository is checked for that path in the current directory. If that path exists, it is returned. Otherwise, the path is returned unchanged.

If the path is an absolute path, it is returned unchanged.

Parameters

$path
A path.
$currentDir
The Puli directory path of the current file.

Returns

string
The resolved path.
