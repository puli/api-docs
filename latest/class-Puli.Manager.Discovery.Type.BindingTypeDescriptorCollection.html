Class BindingTypeDescriptorCollection

A store for binding type descriptors.

Each descriptor has a composite key: * The name of the type. * The package that defines the type.

The store implements transparent merging of types defined within different packages, but with the same type name. If a type is requested for a name without giving a package name, the first type set for that name is returned by Puli\Manager\Discovery\Type\BindingTypeDescriptorCollection::get().
