# parseHomeDirectory( )

Parses environment variables for Puli's home directory.

Parses environment variables for Puli's home directory.

This method scans the environment variables "PULI_HOME", "HOME" and "APPDATA" to determine Puli's home directory: * If "PULI_HOME" is found, that directory is used. * If "HOME" is found, a directory ".puli" is created inside. This variable contains the path of the user's home directory by default on Unix based systems. * If "APPDATA" is found, a directory "Puli" is created inside. This variable contains the path of the application data by default on Windows.

Returns

string
The path to Puli's home directory.

Throws

Puli\RepositoryManager\Api\InvalidConfigException

If no environment variable can be found to determine the home directory.


Puli\RepositoryManager\Api\FileNotFoundException
If the home directory is not found.
Puli\RepositoryManager\Api\NoDirectoryException
If the home directory is not a directory.
