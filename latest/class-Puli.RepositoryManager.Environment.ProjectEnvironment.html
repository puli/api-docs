Class ProjectEnvironment

The environment of a Puli project.

This class contains both global environment information (see Puli\RepositoryManager\Environment\GlobalEnvironment) and information local to a Puli project. It provides access to the project's root directory and the root puli.json of the project.

Use Puli\RepositoryManager\Environment\GlobalEnvironment::getConfig() to access the project configuration.

Creates the project environment.

Creates the project environment.

The passed home directory will be be scanned for a file "config.json". If that file exists, it is loaded into memory. Use Puli\RepositoryManager\Environment\GlobalEnvironment::getConfig() to access the project configuration.

The passed root directory will be be scanned for a file "puli.json". If that file exists, it is loaded into memory. Use Puli\RepositoryManager\Environment\ProjectEnvironment::getRootPackageFile() to access the package file.
