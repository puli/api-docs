Class PathMappingRepository

A development path mapping resource repository. Each resource is resolved at get() time to improve developer experience.

Resources can be added with the method Puli\Repository\AbstractPathMappingRepository::add():

use Puli\Repository\PathMappingRepository;

$repo = new PathMappingRepository();
$repo->add('/css', new DirectoryResource('/path/to/project/res/css'));

This repository only supports instances of FilesystemResource.
