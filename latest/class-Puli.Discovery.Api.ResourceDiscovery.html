Interface ResourceDiscovery

Discovers resources for binding types.

Use Puli\Discovery\Api\ResourceDiscovery::find() to query all bindings bound to a binding type:

$bindings = $binder->find('acme/xliff-messages');

foreach ($bindings as $binding) { foreach ($binding->getResources() as $resource) { $translator->addXlfCatalog( $resource->getFilesystemPath(), $binding->getParameter('translationDomain') ); }
}

Use Puli\Discovery\Api\ResourceDiscovery::getBindings() if you want to retrieve all bindings for a specific resource:

$bindings = $binder->getBindings('/app/trans/errors.fr.xlf');

That method optionally lets you filter the bindings for that resource by their binding type:

$bindings = $binder->getBindings('/app/trans/errors.fr.xlf', 'acme/xliff-messages');
