Creates a new binding.

Creates a new binding.

You can pass parameters that have been defined for the type. If you pass unknown parameters, or if a required parameter is missing, an exception is thrown.

All parameters that you do not set here will receive the default values set for the parameter.

Parameters

The name of the type to bind against.
$parameterValues

The values of the parameters defined for the type.

$uuid

The UUID of the binding. A new one is generated if none is passed.
