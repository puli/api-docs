Class ProjectEnvironment

The environment of a Puli project.

This class contains both global environment information (see Puli\Manager\Api\Environment\GlobalEnvironment) and information local to a Puli project. It provides access to the project's root directory and the root puli.json of the project.

Use Puli\Manager\Api\Environment\GlobalEnvironment::getConfig() to access the project configuration.
