Class ConfigFileStorage

Loads and saves configuration files.

This class adds a layer on top of Puli\RepositoryManager\Config\ConfigFile\Reader\ConfigFileReader and Puli\RepositoryManager\Config\ConfigFile\Writer\ConfigFileWriter. Any logic that is related to the loading and saving of configuration files, but not directly related to the reading/writing of a specific file format, is executed by this class.
