 20: /**
 21:  * A discovery that supports the addition and removal of bindings and types.
 22:  *
 23:  * Binding types have a name and optionally one or more parameters. Binding
 24:  * types can be defined with the {@link define()} method:
 25:  *
 26:  * ```php
 27:  * use Puli\Discovery\Api\Binding\BindingParameter;
 28:  * use Puli\Discovery\Api\Binding\BindingType;
 29:  *
 30:  * $discovery->define(new BindingType('acme/xliff-messages', array(
 31:  *     new BindingParameter('translationDomain', null, 'messages'),
 32:  * ));
 33:  * ```
 34:  *
 35:  * Resources can be bound to these types with the {@link bind()} method:
 36:  *
 37:  * ```php
 38:  * $discovery->bind('/app/trans/errors.*.xlf', 'acme/xliff-messages', array(
 39:  *     'translationDomain' => 'errors',
 40:  * ));
 41:  * ```
 42:  *
 43:  * Use {@link find()} to retrieve bindings for a given type:
 44:  *
 45:  * ```php
 46:  * $bindings = $discovery->find('acme/xliff-messages');
 47:  *
 48:  * foreach ($bindings as $binding) {
 49:  *     foreach ($binding->getResources() as $resource) {
 50:  *         $translator->addXlfCatalog(
 51:  *             $resource->getLocalPath(),
 52:  *             $binding->getParameter('translationDomain')
 53:  *         );
 54:  *     }
 55:  * }
 56:  * ```
 57:  *
 58:  * @since  1.0
 59:  * @author Bernhard Schussek <bschussek@gmail.com>
 60:  */
