public
# setPluginClasses( array $pluginClasses )

Sets the plugin classes.

Sets the plugin classes.

The plugin classes must be fully-qualified class names that implement Puli\Manager\Api\PuliPlugin. If a class is not found or does not implement that interface, an exception is thrown.

The plugin classes must not have required parameters in their constructor so that they can be successfully instantiated. If a constructor has required parameters, an exception is thrown.

Leading backslashes are removed from the fully-qualified class names.

$pluginClasses
The fully qualified plugin class names.
