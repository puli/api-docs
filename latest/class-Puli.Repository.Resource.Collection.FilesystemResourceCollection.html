Class FilesystemResourceCollection

A collection of resources on the filesystem.

The resource collection contains the additional method Puli\Repository\Resource\Collection\FilesystemResourceCollection::getFilesystemPaths() for batch collecting the filesystem paths of all contained resources.

Returns the filesystem paths of all contained resources.

Returns the filesystem paths of all contained resources.
