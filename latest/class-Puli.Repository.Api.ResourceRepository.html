Interface ResourceRepository

Stores Resource objects.

A resource repository is similar to a filesystem. It stores Resource objects, each of which has a path in the repository:

$resource = $repo->get('/css/style.css');

Resources may have child resources. These can be accessed with Puli\Repository\Api\ResourceRepository::listChildren():

$resource = $repo->get('/css');

foreach ($resource->listChildren() as $name => $resource) { // ...
}
