Class RepositoryUpdater

Adds new resources to a resource repository.

You can register resource mappings for insertion by calling Puli\RepositoryManager\Repository\RepositoryUpdater::add(). All registered mappings will be inserted into the repository when you call commit(). This method respects the package order specified in the package graph passed to the constructor.
