Class GlobalEnvironmentImpl

The global environment.

This class contains global environment information. It provides access to Puli's home directory, the global configuration and the global event dispatcher.

Use Puli\RepositoryManager\Environment\GlobalEnvironmentImpl::getConfig() to access the global configuration.

Creates the global environment.

Creates the global environment.

The passed home directory will be be scanned for a file "config.json". If that file exists, it is loaded into memory. Use Puli\RepositoryManager\Environment\GlobalEnvironmentImpl::getConfig() to access the global configuration.
